# Bad Apple Terminal Tool

Have you ever wanted Bad Apple to run right in your terminal?  
No? Oh well, now you can!

## Installation

```bash
# clone the repo
$ git clone https://github.com/infi-defi/badApple.git

# change your working directory
$ cd badApple

# make
$ make install
```

## Usage

```bash
# simply run:
$ badApple
```
